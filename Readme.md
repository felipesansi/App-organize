# organizarApp

Aplicativo para organizar tarefas, compromissos e informações pessoais de forma simples e eficiente.

## Funcionalidades

- Cadastro de tarefas
- Organização por categorias
- Marcação de tarefas concluídas
- Interface intuitiva
